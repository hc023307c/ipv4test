# IP 子網路計算網站工作計畫

## 專案目標
- 建立一個可於瀏覽器操作的子網路計算網站。
- 使用者可輸入：
  - IP 位址
  - CIDR 首碼，例如 `/24`
  - 或子網路遮罩，例如 `255.255.255.0`
- 網站需輸出：
  - 子網路遮罩
  - 網段名稱
  - 廣播位址
  - IP 網段範圍
  - 可用 IP 範圍
- 前端檔案需拆分為：
  - `index.html`
  - `style.css`
  - `app.js`
- 視覺風格方向：
  - 吉普力風
  - 溫暖、自然、手作感
  - 但仍需保有可讀性與操作清楚度
